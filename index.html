Проект деревянного дома «Симфония»

Дом создан для комфортного проживания семьи двух-трех поколений, а также их гостей. Дом, где людей объединяет любовь к природе, свету, воздуху. Дом, где у каждого члена семьи есть свое личное пространство. Идея дома - гармоничное созвучие материала и архитектуры, человека и природы. На него работают как просторная общая терраса, где хозяева и гости могут вместе провести вечер, так и приватная часть террасы с выходом из спальни хозяев. Эту идею поддерживает многофункциональная основная двусветная гостиная. В гостевой части дома предусмотрено все для короткого или более длительного визита гостей. Гостиная с мини-кухней, 2 спальни, собственный санузел с ванной. Дом имеет современную сауну, навес для автомобилей и гараж для катера, который может использоваться для любых хозяйственных нужд. Проект идеально подойдет для вытянутого в длину земельного участка.

      ПОДРОБНЕЕ

      Фундамент

      Монолитный железобетонный свайно-ростверковый с плитой. Высотой от горизонтального уровня земли 500 мм. 
      • Геодезические работы: разбивка осей, нивелировка.
      • Выемка грунта согласно плану фундамента.
      • Укладка Геотекстиль.
      • Устройство буронабивных свай.
      • Устройство песчаной подушки (с учетом послойной виброусадки).
      • Изготовление и монтаж опалубки.
      • Изготовление армопояса продольной арматурой д12 с технологией вязки по СНИП.
      • Заливка монолитной железобетонной ленты ростверка.
      • Заливка плиты.
      • Бетон М 300 (сертифицированный).
      • Заливка бетона через бетононасос.
      • Виброусадка бетона для увеличения прочности.

      Стеновой комплект дома

      • Стеновой комплект дома, размеры в осях согласно планам этажа.
      • Между первым венцом дома и фундаментом прокладывается слой гидростеклоизола для герметизации и отсутствия контакта между деревом и бетоном.
      • Подкладная доска по периметру стен первого этажа сечением 200мм х 50мм по всей длине дома, материал лиственница (естественной влажности). Крепится к бетону при помощи анкеров.
      • Подкладной брус по периметру стен первого этажа сечением 100х200мм, материал хвойных пород (естественной влажности). Крепится к подкладной доске при помощи саморезов.
      • Закладная доска Naturi под все стены в два уровня, материал хвойных пород влажностью 12-14%. Крепится к подкладному брусу при помощи саморезов.
      • По закладной доске прокладывается в два слоя ПСУЛ для герметизации зазоров.
      • Наружные стены дома выполняются по технологии Naturi 300 мм (согласно эскизному проекту), материал хвойных пород влажностью 12-14%. При монтаже стен используются деревянные нагеля и саморезы для стягивания деревянных конструкций с неполной резьбой и просечкой.
      • Внутренние несущие стены выполняются по технологии Naturi 200 мм (согласно эскизному проекту), материал хвойных пород влажностью 12-14%. При монтаже используются саморезы для стягивания деревянных конструкций с неполной резьбой и просечкой.
      • Внутренние перегородки выполняются по карксной технологии толщиной 150мм (согласно эскизному проекту), материал хвойных пород. При монтаже используются саморезы для стягивания деревянных конструкций с неполной резьбой и просечкой. Без утепления и обшивки.
      • Мауэрлат перехода между первым этажом и кровлей выполняется из бруса сечением 100х200мм в два уровня, материал хвойных пород (естественной влажности). Крепится при помощи саморезов.
      • Видовые балки выполняются из бруса сечением 100х300мм согласно проекту, клееная древесина.
      • Вертикальные колоны выполняются из бруса сечением 200х200мм согласно плану этажей, клееная древесина.
      • Фронтоны фасадов зашиваются подшивной доской, материал хвойных пород влажностью 12-14%. Гидроизоляция. Крепится к каркасу при помощи саморезов.
      • Высота стен этажа: 2,9 м.

      Крыша

      • Пиломатериал (стропильная группа 200х50 мм, вентиляционный брусок 50х50 мм, контробрешетка).
      • Покрывной материал кровли (мягкая черепица), крепежи и метизы для стропильной системы, пароизоляционные и гидроизоляционные пленки, теплоизоляция, расходный материал и огнебиозащита.
      • Устройство стропильной группы.
      • Обработка огнебиозащитой стропильной группы и потолочных балок.
      • Настил пароизоляции.
      • Настил теплоизоляции (толщиной 200 мм).
      • Настил гидроизоляции.
      • Установка вентиляционного бруска.
      • Установка контробрешетки.
      • Настил покрывного материала кровли
      • Установка сливной системы.
      • Чердачное перекрытие выполняются из доски сечением 50х200мм (согласно плана этажа) с шагом 0,6 м в осях, материал хвойных пород (естественной влажности).
      • Крепление на опоры бруса.
